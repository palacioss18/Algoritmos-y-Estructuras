# 🔍 Algoritmos de Búsqueda

Esta sección contiene implementaciones de diferentes métodos para encontrar elementos dentro de una estructura de datos.

## 📈 Comparativa de Eficiencia

* **Búsqueda Lineal:** Recorre uno por uno los elementos. 
    * **Complejidad:** $O(n)$
    * **Uso:** Arreglos desordenados o pequeños.
* **Búsqueda Binaria:** Divide el espacio de búsqueda a la mitad en cada paso.
    * **Complejidad:** $O(\log n)$
    * **Requisito:** El arreglo **DEBE** estar ordenado previamente.

---
*Nota: La eficiencia de la búsqueda binaria es fundamental en bases de datos y sistemas de alto rendimiento.*
