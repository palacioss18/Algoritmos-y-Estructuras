# 📊 Algoritmos de Ordenamiento

Aquí se encuentran las implementaciones para organizar datos de manera ascendente o descendente, analizando su consumo de tiempo y memoria.

## 📂 Clasificación

### Algoritmos Simples (Ideales para listas pequeñas)
* **Bubble Sort:** Intercambio de elementos adyacentes. ($O(n^2)$)
* **Insertion Sort:** Inserta cada elemento en su posición correcta. ($O(n^2)$)
* **Selection Sort:** Busca el mínimo y lo coloca al principio. ($O(n^2)$)

### Algoritmos Avanzados (Eficientes para grandes volúmenes)
* **Quick Sort:** Basado en la técnica de "Divide y Vencerás" usando un pivote. ($O(n \log n)$)
* **Merge Sort:** Divide el arreglo en mitades y luego las combina ordenadamente. ($O(n \log n)$)

---
> "Elegir el algoritmo correcto depende del tamaño de los datos y de las restricciones de memoria del sistema."
