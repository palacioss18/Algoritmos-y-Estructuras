# 🧠 Algoritmos y Estructuras de Datos en C++

Este repositorio está dedicado al estudio, implementación y análisis de eficiencia de estructuras de datos y algoritmos fundamentales utilizando **C++**. 

> "La mala programación es código que funciona. La buena programación es código que además es eficiente y fácil de mantener."

---

## 🚀 Contenido del Repositorio

### 🏗️ Estructuras de Datos
* **Lineales:** Pilas (Stacks), Colas (Queues), Listas Enlazadas (Simples y Dobles).
* **No Lineales:** Árboles Binarios de Búsqueda (BST), Grafos.

### ⚡ Algoritmos
* **Ordenamiento:** Bubble Sort, Insertion Sort, QuickSort y MergeSort.
* **Búsqueda:** Búsqueda Lineal y Búsqueda Binaria $O(\log n)$.
* **Recursividad:** Resolución de problemas mediante lógica recursiva.

---

## 📊 Análisis de Complejidad
En cada implementación busco optimizar el uso de recursos, documentando la **Notación Big O**:
* **$O(1)$**: Tiempo constante.
* **$O(n)$**: Tiempo lineal.
* **$O(n^2)$**: Tiempo cuadrático (evitar en grandes volúmenes de datos).
